# G4TX-G4TG-Tool

Rebuilds .GNF from .g4tx/.g4tg textures from PS4 versions of Yo-Kai Watch 4++ and Yōkai Wotchi Jam: Yōkai Gakuen Y ~N to no Sōgū~ (Y School Heroes)


Usage: 


python ps4g4tool.py extract title00_01.g4tx title00_01.g4tg title00_01.gnf


python ps4g4tool.py rebuild title00_01.g4tx title00_01_edited.gnf title00_01.new.g4tx title00_01.new.g4tg

python ps4g4tool.py batch_extract
